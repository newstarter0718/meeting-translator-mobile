# Meeting Translator Mobile PWA

Real-time meeting transcription & translation powered by **Gemini AI**.

- 🎙️ Tap to record, auto-sends on silence
- 🌐 Supports EN / VN / TH / KH / Cantonese → Traditional Chinese & more
- 📥 Export as .txt
- 🔒 API Key stored locally on-device only

## Usage
Open the app URL in your mobile browser, enter your [Gemini API Key](https://aistudio.google.com), and tap record.
